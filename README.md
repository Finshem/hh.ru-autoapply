# hh.ru Auto-Apply

Простейший скрипт на Python + Playwright для автоматического отклика на вакансии на hh.ru.

## Возможности

- Автоматический сбор вакансий из сохранённого поиска и отклик на них
- Подстановка сопроводительного письма из `cover_letter.txt`, если оно требуется
- Определение суточного лимита откликов (200/24ч) и остановка без ошибок
- Учёт откликов (`applied.txt`) и неудачных попыток (`failed.json`)
- Скриншоты для отладки, если кнопка отклика не найдена

## Установка

```bash
git clone https://github.com/Finshem/hh.ru-autoapply.git
cd hh.ru-autoapply
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
playwright install chromium
```

## Первый запуск 
В `apply.py` нужно заменить `search_url` на ссылку рекомендуемых вакансий для Вашего резюме, либо иную с целевыми вакансиями для отклика.

(Опционально) для отклика на вакансии в которых необходимо сопроводительное письмо нужно добавить текст универсального сопроводительного письма в `cover_letter.txt`.

## Запуск

```bash
python apply.py
```

Откроется браузер — нужно вручную войти в аккаунт hh.ru, затем нажать Enter в терминале для продолжения.

## Настройки (переменные окружения)

| Переменная | По умолчанию | Описание |
|---|---|---|
| `HH_PAUSE_MIN` | 10 | мин. пауза между откликами (сек) |
| `HH_PAUSE_MAX` | 15 | макс. пауза между откликами (сек) |
| `HH_LIMIT` | 200 | лимит откликов за запуск |
| `HH_MAX_RETRIES` | 2 | сколько раз повторять неудачную вакансию |
| `HH_COVER_LETTER_FILE` | cover_letter.txt | путь к файлу с письмом |
